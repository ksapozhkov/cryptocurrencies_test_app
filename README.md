# Cryptocurrencies Test App

### Description:
This is a test Android app that displays current price in USD for a few cryptocurrencies. 

### Technologies:
* Language: Kotlin
* Presentation pattern: MVI (Elmslie)
* UI: Compose
* Network: Retrofit
* Data persistence: Room
